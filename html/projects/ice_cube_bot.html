Ice Cube Bot

This is a discord bot that was created for Snow Codes Bot Jam! It is a winter themed PVP bot that allows users to fight winter monsters inorder to gain coins. This can be used to upgrade attack and defense stats making players more powerful. Players were able to challenge other players to duels where the winner takes the loser's money. The bot is coded in Python and uses local sql datebases to store user data. The most challenging part was figuring out the idea and logic behind the battle sequencing. However, in the end most of the functions seemed to work fine and the Bot ended up getting 7th place overall. I really enjoyed this project because it increased my knowlege of Python and how to manage user data.
